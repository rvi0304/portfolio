ResumeIQ

OCR + NLP Resume Analyzer

Developed ResumeIQ, a web application that automates the process of analyzing and comparing multiple resumes (PDF/TXT) to identify the most relevant candidate for a specific job role. Users can upload up to 10 resumes, which are extracted using OCR-enabled methods. The system then performs keyword-based and fuzzy matching to highlight the best-suited resume, allows downloading it as a PDF, and stores all uploaded files securely in the system.

Technologies Used:

  • Backend: Python, Flask (with CORS support)
  • Text Extraction OCR: pdfplumber, pdf2image, pytesseract
  • NLP Matching: NLTK, TfidfVectorizer, FuzzyWuzzy
  • Functionality: Multi-resume upload, keyword-based best match selection, view best resume in same page, file saving (uploads folder)
  • Deployment: Flask API with CORS support
